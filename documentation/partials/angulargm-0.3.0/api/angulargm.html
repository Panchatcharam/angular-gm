Module: angulargm

Module for embedding Google Maps into AngularJS applications.

API Documentation

See...

  • gmMap for usage of the gm-map directive

  • gmMarkers for usage of the gm-markers directive

  • gmInfoWindow for usage of the gm-info-window directive

  • angulargmContainer if you need to run custom configuration on the map, e.g. add new map types

  • angulargmDefaults to override the default map options

  • angulargmUtils to use utility functions for LatLng and LatLngBounds
